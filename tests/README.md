# Tests

This directory is reserved for integration and end-to-end tests that span application boundaries.
Frontend unit and component tests may live beside their source files under `frontend/src/` when a
test runner is introduced.

Priority coverage:

1. Email and phone authentication.
2. Company onboarding and role assignment.
3. Buyer and supplier route isolation.
4. Cross-tenant data access prevention.
5. RFQ assignment and quotation authorization.
